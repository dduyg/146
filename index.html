Vaporwave Art Gallery

This is a web application that dynamically generates and displays vaporwave-style artwork using the Unsplash API. The frontend is built using HTML, CSS, and JavaScript. The artwork is generated dynamically by making requests to the Unsplash API, fetching random images tagged with 'vaporwave'. The generated artwork is displayed in the gallery section below. You can click the "Generate Artwork" button to fetch new artwork and add it to the gallery.
